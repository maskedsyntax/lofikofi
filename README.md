# Lofikofi

Lofikofi is a desktop first coworking space for people who want a calm, atmospheric place to work without distractions. It is designed as a digital room you open before you start working, not just another productivity tool in a tab.

## What you get right now

### Ambient soundscapes

- A layered ambient mixer with multiple scenes such as rain, cafe, forest, fireplace and more
- Per layer volume control so you can blend a custom scene that matches your energy
- Local sounds bundled with the app so everything works offline

### Focus timer

- A compact glass like focus panel that stays out of the way
- Pomodoro mode with a simple 25 minute focus and 5 minute break cycle
- Deep work mode for longer, interruption free sessions
- Clear status and time remaining, tuned for quick glances rather than dashboards

### Desktop first experience

- Runs as a Tauri desktop app so it feels like a native workspace
- Browser based preview for fast iteration and quick checking of the UI
- All core behavior works offline once installed

## What Lofikofi is aiming for

Lofikofi is meant to be a place where you sit down, start a scene, start a focus session and let the rest of the world get quieter. The long term goal is to combine soundscapes, a light focus system and a very small set of planning tools into one consistent environment rather than a collection of separate apps.

It is not trying to replace project management software or team tools. It is intentionally narrow so that opening it always feels the same: quiet, familiar, and ready for focus.
